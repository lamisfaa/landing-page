# Tutorings Hero Section Specification

This document defines the detailed hero section direction for the Tutorings lead-generation landing page. It is documentation only. Do not implement until the page direction is approved.

## Hero Role

The hero is the first major storytelling moment of the landing page. It should communicate Tutorings' promise clearly, establish the premium SaaS/editorial tone, and introduce the visual metaphor of scattered learning becoming structured progress.

## Page Entry

The page starts with a top navigation bar before the hero content.

Navigation structure:

- Logo on the left.
- Primary CTA on the right.
- Minimal navigation treatment.

The Tutorings logo will be provided separately and should be used in the navigation when available.

## Hero Layout

The hero content should be centered.

Required order:

1. Navigation at the top.
2. Centered headline.
3. Supporting paragraph below the headline.
4. Primary CTA and secondary CTA centered below the supporting paragraph.
5. Large immersive visual occupying the bottom half of the hero.
6. Trust logos below the hero.

## Headline

Headline text:

> Master English with confidence.

Headline requirements:

- Centered.
- Very large.
- Bold.
- Maximum two lines.
- Premium and editorial in tone.
- Should be the dominant element in the first viewport.

The headline should feel confident and direct. It should not be surrounded by too much explanatory text.

## Supporting Paragraph

The supporting paragraph sits directly below the headline.

Content direction:

- Explain that Tutorings helps learners move from scattered English practice to structured progress.
- Mention English courses, IELTS preparation, and private tutors.
- Keep it narrow and concise.

Paragraph requirements:

- Centered.
- Narrow line length.
- No dense explanation.
- Should support the headline, not compete with it.

## Calls To Action

Buttons sit centered below the supporting paragraph.

Primary CTA:

- Start Learning

Secondary CTA:

- Book a Consultation

CTA behavior:

- Primary CTA should be visually dominant.
- Secondary CTA should be quieter but still clear.
- Hovering the CTA should subtly brighten nearby visual elements.
- The hover effect should feel premium and understated.

## Whitespace

The hero should use lots of whitespace.

The top text area should feel calm, spacious, and uncluttered. The large headline, narrow paragraph, and centered buttons should have enough breathing room to create a premium landing-page opening.

Avoid:

- Crowding the headline with badges or excessive trust claims.
- Adding too many CTAs.
- Placing cards around the text that make the hero feel busy.

## Main Visual

The large visual occupies approximately the lower 45-50% of the hero.

Visual requirements:

- Should sit in the bottom half of the hero.
- Should extend beyond the content container.
- Should feel immersive rather than boxed in.
- Should not look like a framed card, dashboard screenshot, or generic illustration.

The visual should communicate the transformation from scattered learning to structured progress.

## Visual Metaphor

Gray paths represent scattered learning.

Tutorings logo represents the turning point.

Blue paths represent structured progress.

Narrative:

1. Gray paths start as scattered, disconnected routes.
2. The paths slowly move toward the center.
3. The Tutorings logo acts as the organizing point.
4. Blue paths emerge from that point as clearer, more structured progress.

The metaphor should feel simple and premium. It should not become overly literal or visually noisy.

## Visual Elements

Potential surrounding learning fragments:

- YouTube.
- PDFs.
- Notes.
- Apps.
- AI tools.
- Tutors.
- IELTS tasks.
- Vocabulary lists.
- Grammar rules.
- Exam deadlines.

These can be represented as icons, labels, or subtle text fragments, but they should remain secondary to the headline and path visual.

## Motion Direction

Motion should be subtle.

Gray paths:

- Slowly move toward the center.
- Should suggest scattered learning becoming organized.

Blue paths:

- Slowly pulse.
- Should suggest structure, progress, and confidence.

CTA hover:

- Subtly brightens nearby elements.
- Should feel connected to the visual system.

Avoid:

- Dramatic movement.
- Fast path animation.
- Excessive pulsing.
- Distracting loops.
- Any animation that competes with the headline.

## Trust Logos

Trust logos appear below the hero.

Purpose:

- Reinforce credibility after the main promise.
- Support the user's confidence before entering the problem section.

Possible trust signals from existing Tutorings content:

- IDP.
- Cambridge One.
- Trustpilot.
- Monsha'at.
- Misk Launchpad.
- Tabby.
- Tamara.

Treatment:

- Calm and minimal.
- Visually secondary.
- Should not crowd the hero.

## Transition To Problem Section

The hero should naturally transition into the blue problem section.

The gray path motif can continue or echo into the next section, where scattered icons and text become more prominent around the large centered problem headline.

This transition should make the next section feel like a deeper look at the problem introduced by the hero visual.
